SegWit2X Simple Countdown Timer

Bitcoin is forking in November based on the SegWit2X code on block height 494784. This is a simple count down timer that calculates the targeted fork block height minus the current block height 488560 (when this was written) multiplied by the Bitcoin 10 minute block interval times. 

Timer Calculation: (494784 - 488560) * (10) * (60)
SegWit2X Simple Countdown Timer was created and is maintained by CoinSalad.com.
